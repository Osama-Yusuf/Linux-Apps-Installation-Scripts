# Linux Apps Installation Scripts

## Downlaod any of these scripts and give it execution permission using this command with this command:

### AnyDesk

```
curl "https://raw.githubusercontent.com/Osama-Yusuf/Linux-Apps-Installation-Scripts/main/anydesk.sh" -o anydesk.sh && chmod +x anydesk.sh
```
